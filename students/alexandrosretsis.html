TAGS USED

    Asside the most common tags, special tags i used in my page as follows:

  • <Footer> tag! This was used to highlight information at the begining or end of a page/s.

  • <span> tag! this was used to style and group elements, instead of the classic div tag, the only difference really is that the span is an inline element.

  • <section> tag! This was used to define and seperate different sections in my pages.

  • <nav> tag! This was used to create the navigation bar for this personal page only, whereas in the index page a <ul> tag was used for the navbar.

  • <script> tag! we used this not only to import special fonts, but also to to write and implement java script code for the enrichment of our page

  • And lastly the <button> tag! although its quite simple to use, its also very effective if used right, we used this to make clickable buttons that link to other areas of our website.
